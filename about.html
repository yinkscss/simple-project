About Console Inspector

A real-time runtime analysis tool designed to inspect local files and render executing script choices.

Overview

The Console Inspector & Name Decider is an interactive web-based sandbox built to analyze local scripts. It reads and monitors variables, overrides standard outputs to pipe them to a customized terminal environment, and triggers analysis on command.

Core Architecture

Variable Syncing

Monitors global workspace environment variables inside executing scripts to dynamically reflect their values on the dashboard cards.

Console Hooking

Overrides the browser's standard console.log mechanism, ensuring all internal script outputs are piped directly into the styled UI terminal.

Hot Re-evaluation

Enables developers to execute comparison logic instantly on click without full page refreshes, optimizing the runtime feedback loop.

Responsive Glassmorphism

Employs modern CSS properties like backdrop-filter and radial color blur structures, styled adaptively for all screen dimensions.
